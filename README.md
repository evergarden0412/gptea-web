## gptea 프로젝트  
GPT API를 사용하여 GPT를 리디자인한다.  
메세지를 즐겨찾기로 저장할 수 있는 **Scrap** 기능을 추가하고, 추후에 메세지에서 특정 문구를 형광펜을 칠하는 등의 텍스트꾸미기를 할 수 있도록 **Highlight**기능을 추가할 예정이다.
